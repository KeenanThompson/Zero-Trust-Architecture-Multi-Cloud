# Okta SSO and SCIM Integration

SSO Integrations Completed:
- Okta → Entra ID (federation)
- Entra ID → Okta
- Okta → AWS (SAML)
- Okta SCIM → AWS (user provisioning)

Configured SAML Elements:
- Assertion Consumer Service (ACS)
- Entity IDs
- Application certificate (Base64)
- Attribute mappings:
  - email
  - username
  - role
  - groups

Group Assignments:
- AWS Admin
- AWS ReadOnly
- AWS DevOps
