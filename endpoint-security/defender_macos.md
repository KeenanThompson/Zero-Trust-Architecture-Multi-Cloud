# Microsoft Defender for Endpoint on macOS

## Onboarding Steps

1. Downloaded the onboarding package for macOS from the Defender for Endpoint portal.
2. Installed the agent on the macOS endpoint.
3. Completed registration so the device appears in the Defender portal.

## Capabilities Validated

- Real-time protection status visible in the portal.
- Device health and onboarding state confirmed.
- Basic alert or test detection generated and observed.

## Relationship to Other Endpoints

- Windows and Linux endpoints are primarily represented through:
  - Defender where licensing and edition allow.
  - CrowdStrike sensor coverage and local hardening.

This file focuses on Defender for Endpoint on macOS.
